# Finish Line Detail Website

Free static website starter for GitHub Pages.

## What to edit first

1. In `index.html`, replace `finishlinedetail@example.com` with your real business email.
2. Replace the placeholder `FL` logo with your real logo image when ready.
3. Add real before/after images to the gallery section.
4. Update services and wording as your packages become finalized.

## How to publish for free on GitHub Pages

1. Create a free GitHub account.
2. Create a new repository named `finish-line-detail` or `finishlinedetail`.
3. Upload these files: `index.html`, `styles.css`, `script.js`, and `README.md`.
4. Go to repository Settings > Pages.
5. Under “Build and deployment,” choose “Deploy from a branch.”
6. Select the `main` branch and `/root`, then save.
7. GitHub will give you a free website link after it deploys.

## Optional later upgrade

Buy a custom domain such as `finishlinedetailnc.com` and connect it to GitHub Pages.
